Game Developer Club

Game developer club is hosted every Saturday at Spokane Public Library. The next 2 events are:

  • Saturday noon to 4pm @ South Hill Branch
  • Saturday noon to 4pm @ Hillyard Branch

The goal of this group is to work together to learn how to become better game developers, network, collaborate, and learn. We focus on an overall group project for ideas but are welcome for people to show up with their own projects to work on.
