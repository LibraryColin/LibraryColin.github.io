Game Developer Club

This site is a work in progress...

Game developer club is hosted every Saturday at Spokane Public Library. The next 2 events are:

  • Saturday noon to 4pm @ South Hill Branch - March 2nd, 2024
  • Saturday noon to 4pm @ Hillyard Branch - March 9th, 2024

The goal of this group is to work together to learn how to become better game developers, network, collaborate, and learn. We focus on an overall group project for ideas but welcome people to show up with their own projects to work on as well.
